# Debug Scripts

This directory contains ad-hoc scripts used during development and debugging.

## Notes

- These scripts are **not** part of the official test suite.
- Official tests live under `test/`.
- Scripts here may import internal modules from `lib/src/` and can be unstable.

## Run

From `g2pkk_dart/`:

```bash
dart run debug/<script_name>.dart
```

Example:

```bash
dart run debug/test_trace.dart
```
